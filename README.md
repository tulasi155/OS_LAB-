# OS Lab Programs

This repository contains a collection of Operating Systems lab programs along with their outputs.

## Contents:
- Unix Commands
- Shell Programming
- System Calls
- I/O System Calls
- Inter Process Communication (IPC)
- Semaphores and Process Synchronization
- CPU Scheduling Algorithms (FCFS, SJF)
- Banker’s Algorithm
- Memory Allocation Techniques (First Fit, Worst Fit)
- Page Replacement Algorithms
- Disk Scheduling Algorithms

## Features:
- Well-structured folder organization
- Each program includes output
- Easy to understand and run

## Usage:
Clone the repository and run the programs using a C compiler or Linux shell environment.
